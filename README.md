🎉 Event Organizer Web App

A simple and responsive Event Organizer Web Application built using HTML, CSS, and JavaScript. This project allows users to create, manage, and explore events in an easy way.

---

🚀 Features

- 📅 Create and manage events
- 📊 Dashboard to view all events
- 🎟️ Ticket management system
- 🔍 View detailed event information
- 📱 Responsive design (mobile + desktop)

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript
